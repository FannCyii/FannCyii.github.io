#Hexo# 发布
第一次使用hexo发布文章到github上时，需要执行npm install hexo-deployer-git --save 命令安装git发布组件，再执行hexo d即可发布。注意需要配置好config.yml文件中的deploy

#Hexo 新增文章
使用hexo new "new title name" 可以生成新的md文件，在source/_post目录下可以看到。之后hexo g编译生成今天文件，hexo s本地查看web效果，hexo d发布即可
